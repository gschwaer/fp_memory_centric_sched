This example shows ERIKA3 running as Jailhouse inmate with two tasks writing in mutual exclusion on the serial interface.
The specific platform can be specified in the OIL file through the SOC_DATA field. For example, one of the following settings:

   SOC_DATA = NVIDIA_TEGRA_X1;
   SOC_DATA = XILINX_ZYNQ_ULTRASCALE_PLUS;
